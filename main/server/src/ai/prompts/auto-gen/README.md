# auto-gen

generated files. do not hand-edit.

`web-routing.json` — the app's route table (pattern + description per route), derived from the dashboard's self-registered routes (the `defineRoute` calls collected by `routeDefs()`). regenerated by `scripts/build-scripts/build-web-routes-script.ts` on every build (`npm run sync:web-routes`, wired into `npm run chain:build` via `chain:sync:all`). add or remove a route in the dashboard and it reflects here automatically. the `dashboard-web-routing` ai provider reads this file at startup.
